# vue-test-utils-casebycase

vue-test-utils (for vue.js 2) Component Test Example

Guide (korean)  
https://velog.io/@sonagi784/Vue-컴포넌트-단위테스트-case-by-case

## Project setup

```
npm install
```

## Run unit tests

```
npm run test:unit
```

## Test on save

```
npm run test:unit-watch
```

## Launch dev server

```
npm run serve
```
