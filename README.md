vue-test-utils (for vue.js 2) Component Test Example

Guide (korean)  
https://velog.io/@sonagi784/Vue-Test-Utils-컴포넌트-테스트-작성-요령

## Project setup

```
npm install
```

## Run unit tests

```
npm run test:unit
```

## Test on save

```
npm run test:unit-watch
```

## Launch dev server

```
npm run serve
```
